#  IPL Match Analysis & Prediction

Exploratory data analysis on Indian Premier League 
data covering matches from 2008 to 2024.

## covered Topics
- Data cleaning and preprocessing
- EDA — team performance, toss effect, top players, venue stats
- Match winner prediction using Random Forest

## Tools used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook

## Dataset
IPL Complete Dataset from Kaggle
